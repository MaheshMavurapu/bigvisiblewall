# Build and Run Tests #

  1. [Checkout](http://code.google.com/p/bigvisiblewall/source/checkout) the code
  1. Run ` ant `

# Create a distribution #

  1. [Checkout](http://code.google.com/p/bigvisiblewall/source/checkout) the code
  1. Run ` ant dist `. The created zip file will be in the ` dist ` folder, and will contain all dependencies.
